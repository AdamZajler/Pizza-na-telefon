
# "Mobilna" aplikacja Pizza na telefon

Aplikacja umożliwiająca przeglądanie i zamawianie pizz przez telefon

## Funkcjonalności

- Zamawianie pizzy oraz napojów
- Wybór płatności kartą lub gotówką
